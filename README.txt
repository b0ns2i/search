Данный код выполяет алгорит по поиску директорий. 
На вход может поступить:
C:\Program Files\*\doc\logs.log
C:\Program Files\**\doc\logs.log
C:\Program Files\*\doc\*.log 
C:\Program Files\**\doc\*.log

Program Files и расширение файла можно менять. Искомый файл должен находиться в папке, который будет известен программе, т.е. имя doc можно менять, но не убирать вовсе. 

По завершению заботы программы будет выведено сообщение "Программа завершила работу!". Поиск может идти 30+ сек, это зависит от того, как глубоко спрятана папка.
